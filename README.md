### 1、用处
一键生成Mybatis Mapper文件、Dao文件，Service ， Controller层增删改查代码
### 2、介绍
本工具基于模板配置而成，可自定义模板，可以生成自己的Controller/Service/Dao/XML文件，比如您想要在类中自定义一些注解，修改模板即可
